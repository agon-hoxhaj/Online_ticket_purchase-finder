Ky projekt është një aplikacion web i ndërtuar në PHP, që lejon përdoruesit të shfletojnë evente dhe të blejnë bileta. Sistemi momentalisht nuk përdor databazë, por ruan të dhënat në file .txt.

Projekti përfshin:

Menaxhim të përdoruesve (register/login)
Shfaqje të eventeve
Zgjedhje të ulëseve
Blerje të biletave
Panel të admini
